Project Description

In this project, I explored a dataset of mobile phone sales to answer the question: "Which smartphone brand offers the best specifications for the lowest price?"

By cleaning, analyzing, and visualizing the data using Python, I was able to uncover insights about value-for-money trends in the smartphone market. The analysis included specifications such as RAM, storage, camera quality, battery life, and price comparisons across brands.
